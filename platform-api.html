Endpoints

Core resources.

All the data you need, through clean and consistent endpoints.

GET /api/v1/products

List all products with filtering, pagination, and branch-level stock data. Supports barcode lookup.

Response
{
  "status": "success",
  "data": {
    "products": [...],
    "pagination": { "page": 1, "total": 2847 }
  }
}
POST /api/v1/sales

Create a new sale transaction. Supports multi-item carts, discounts, payment splitting, customer linking, and receipt generation.

Request Body
{
  "branch_id": 3,
  "items": [
    { "product_id": 1042, "qty": 2, "discount": 0 }
  ],
  "payment_method": "cash",
  "customer_id": 584
}
GET /api/v1/inventory/{branch_id}

Get real-time stock levels for a specific branch. Includes low-stock alerts, reorder points, and category breakdown.

Response
{
  "branch": "Downtown Store",
  "total_items": 4231,
  "low_stock": 23,
  "categories": [...]
}
GET /api/v1/reports/revenue

Revenue reports with date ranges, branch filtering, and comparison periods. Supports daily, weekly, and monthly aggregation.

Response
{
  "period": "2025-06",
  "revenue": 847320.50,
  "transactions": 3421,
  "avg_basket": 247.66
}
PUT /api/v1/customers/{id}

Update customer profile, loyalty points, group membership, and contact information. Includes purchase history linking.

Request Body
{
  "name": "Ahmed Hassan",
  "phone": "+20 10 1234 5678",
  "group": "VIP",
  "loyalty_points": 2450
}
POST /api/v1/webhooks

Register webhook endpoints for real-time event notifications. Supports sale completed, stock alert, branch sync, and customer events.

Request Body
{
  "url": "https://your-app.com/hooks",
  "events": [
    "sale.completed",
    "stock.low",
    "branch.synced"
  ],
  "secret": "your_webhook_secret"
}
Developer Experience

Built for developers.

Scoped API Keys

Create keys scoped to specific resources, branches, and permission levels. Rotate keys without downtime.

HMAC Authentication

SHA-256 HMAC request signing ensures data integrity and prevents replay attacks. Every request is verified.

Consistent JSON

Every endpoint returns structured JSON with predictable status codes, error messages, and pagination metadata.

Rate Limiting

Transparent rate limits with headers showing remaining quota. Burst allowances for high-throughput integrations.

Webhook Events

Subscribe to 15+ event types. Events are retried with exponential backoff and include delivery logs for debugging.

Versioned & Documented

Full OpenAPI 3.0 specification. Interactive docs, code samples in cURL/Python/Node, and changelog for every version.
