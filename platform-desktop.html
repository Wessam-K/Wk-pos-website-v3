Desktop Features

Full power, no connection.

Everything the web app can do — plus native hardware integration and zero internet dependency.

True Offline Operation

Embedded SQLite database means no server dependency. Sales, inventory, customers — everything stored locally and securely.

LAN Sync

WebSocket-based synchronization between machines on the same network. Main branch becomes the sync server — clients connect seamlessly.

Hardware Integration

Direct connection to thermal printers, barcode scanners, weighing scales, and cash drawers through the native desktop environment.

One-Click Install

Windows installer sets up everything — application, database, and background sync service. Under 100 MB total. No IT department needed.

Background Sync

When internet is available, data syncs to the cloud silently in the background. Nothing interrupts the cashier's workflow.

Conflict Resolution

Smart merge algorithms handle data conflicts when branches reconnect. Last-write-wins with full audit trail for every change.
